# Configuration Guide

### Image name for background
The filename of the image you want to use for the background. Place your images in the `user_files/background` folder. Use "random" to pick a random image from the folder.

### Image name for gear
The filename of the image you want to use for the gear icon. Place your images in the `user_files/gear` folder. Use "random" to pick a random image from the folder.

### Opacity
Set the opacity of the background image (0.0 to 1.0).

### Visibility
- **Show in reviewer**: Enable or disable the background in the reviewer screen.
- **Show in toolbar**: Enable or disable the background in the top and bottom toolbars.

### Layout
- **Attachment**: `fixed` or `scroll`.
- **Size**: `cover` or `contain`.
- **Position**: Adjust where the image is centered.
- **Scale**: Scale the background image.

### Colors
Set background colors for the main screen and toolbars (useful when opacity is less than 1).
