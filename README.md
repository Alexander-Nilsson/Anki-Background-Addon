# Custom Background and Gear Icon

This Anki add-on allows you to customize the background image and the gear icon in your Anki workspace.

## Features
- Set a custom background image for the main screen, reviewer, and toolbars.
- Customize the gear icon.
- Adjust background opacity for both main and reviewer screens.
- Change background colors for toolbars and main screen.
- Supports multiple image formats (JPG, PNG, GIF, WEBP, etc.).
- Randomize background and gear icons from your custom folders.

## How to use
1. Open the settings from `Tools` -> `Custom Background and Gear Icon`.
2. Select your desired background image and gear icon.
3. Place your custom images in the `user_files/background` and `user_files/gear` folders.
